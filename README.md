# 🍔 Food Delivery App

A modern and responsive **Food Delivery Mobile Application** built using **React Native**, **Expo**, **TypeScript**, and **React Navigation**.

The app includes:
- Authentication Flow
- Bottom Tab Navigation
- Drawer Navigation inside Profile
- Modern Dark UI
- Orders Management
- Smooth Navigation Experience

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/intekhabx/eatly-food-app.git
```

---

## 2️⃣ Navigate to Project Directory

```bash
cd food-delivery-app
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

# 📦 Required Packages

## React Navigation

```bash
npm install @react-navigation/native
npm install @react-navigation/stack
npm install @react-navigation/bottom-tabs
npm install @react-navigation/drawer
```

---

## React Native Dependencies

```bash
npm install react-native-screens
npm install react-native-safe-area-context
npm install react-native-gesture-handler
npm install react-native-reanimated
```

---

## Expo Dependencies

```bash
npx expo install expo-linear-gradient
npx expo install @expo/vector-icons
```

---

# ▶️ Running the Application

## Start Expo Server

```bash
npx expo start
```
